回退N协议

创建

while(等待事件){

  if( 帧到达 ）

    if( 该帧为期望接收的帧 )

      接收该帧

      将窗口下限移至第一个未接收的帧处

    else 丢弃该帧

    发送序号为窗口下限序号-1帧的确认帧

do{

  依次发送窗口内所有的帧

  为每个帧设定计时器

  if（ 规定时间内收到期望的确认帧 )

    发送窗口滑动至该帧后

    停止窗口外帧的计时器

  else if ( 期望帧的计时器超时 ){

    重发超时帧及其后的N帧,并重设计时器

  else if (收到非期望确认帧)丢弃该帧

}while ( 还需继续发送帧 )
